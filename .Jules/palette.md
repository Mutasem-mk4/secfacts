## 2024-05-03 - Terminal tabwriter Alignment and ANSI Codes
**Learning:** When applying ANSI color codes in terminal output formatted with `text/tabwriter`, the alignment completely breaks unless `\xff` wrapping is used around the ANSI codes and the `tabwriter.StripEscape` flag is unconditionally passed to the tabwriter. Otherwise, the tabwriter counts the hidden escape codes as visible characters, ruining the table layout. Emojis also help significantly in creating a structured, accessible visual hierarchy for severity outputs.
**Action:** When updating CLI output using tabwriter, unconditionally enable `tabwriter.StripEscape`. Conditionally apply `\xff` wrappers around ANSI escape sequences only when outputting to a TTY device. Use emojis for colorblind-friendly severity differentiation.
