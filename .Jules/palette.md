## 2024-05-08 - Formatting ANSI Color Codes with tabwriter
**Learning:** When using Go's `text/tabwriter` to display aligned columns in the terminal, ANSI color escape sequences disrupt the column width calculations, leading to misaligned output. The `tabwriter.StripEscape` flag exists specifically to handle this, requiring escape sequences (or sections of text containing them) to be wrapped with the `\xff` byte. The tabwriter calculates widths ignoring the `\xff`-wrapped content, and strips the `\xff` bytes before outputting the final aligned text to the terminal.
**Action:** Always enable `tabwriter.StripEscape` when printing colorized output to a tabwriter and wrap the terminal color codes in `\xff` characters (e.g., `\xff\x1b[31m\xff`) to preserve alignment while maintaining visual hierarchy in the CLI.
